# java-swing
This repository presents the design and implementation of a Scientific Calculator application developed using the Java Swing framework. The project demonstrates the application of object-oriented programming principles, event-driven architecture, and graphical user interface (GUI) design within the Java programming environment.

The system is structured around a modular class design, where the main class extends the JFrame component to create the application window and implements the ActionListener interface to handle user interactions. The program utilizes standard Java libraries, including Swing and AWT for interface construction, and Math and DecimalFormat for performing and formatting mathematical computations.

The calculator supports a range of functionalities, including basic arithmetic operations, trigonometric functions (with both degree and radian modes), logarithmic operations, exponentiation, and memory-based calculations. It also provides real-time input handling and dynamic updates of both the expression and result display fields.

From a software engineering perspective, the project illustrates key concepts such as encapsulation of variables, method abstraction for modular design, and inheritance for extending graphical components. Additionally, it incorporates custom rendering of user interface elements, demonstrating advanced GUI development techniques using Graphics2D.

Although the current implementation evaluates operations sequentially rather than supporting full expression parsing, it establishes a strong foundation for future enhancements. Overall, the project serves as both a learning tool for Java GUI programming and a practical example of integrating mathematical computation within an interactive software system.
